# Die-Again

A small Unity platformer demo developed in 15 days.

## Features

- Character movement
- Jump system
- Camera follow
- Multiple levels
- Home menu UI
- Level selection system
- Teleport portal system
- Platform obstacles
- UI buttons and transitions
- Basic platformer gameplay

## Built With

- Unity
- C#

## Project Status

Currently in demo development stage.
This demo currently includes only Level 1 and Level 2.
More levels, mechanics, and gameplay features are planned for future updates.

## Future Plans

- Add enemy AI
- Add sound effects
- Improve animations
- Add more levels
- Improve UI design
